1.1 Dasturlash nima va nima uchun aynan Python?

Dasturlash — bu kompyuterga nima qilish kerakligini buyruqlar orqali tushuntirish san'ati. Tasavvur qiling, sizda juda aqlli lekin faqat buyruqni tushunadigan yordamchingiz bor. Dasturlash — shu yordamchiga tushunarli tilda gapirishdir.

💡 Oddiy misol: Restoranda ofitsiantga "1 ta pitsa va 2 ta kola bering" deysiz. Bu — buyruq. Dasturlashda ham xuddi shunday: kompyuterga "ekranga 'Salom' deb yoz" deymiz.

Nima uchun Python?

  • ✅ O'rganish oson: Ingliz tiliga eng yaqin dasturlash tili. Sintaksis sodda va tushunarli.
  • ✅ Backend yetakchisi: Django, Flask, FastAPI kabi frameworklar orqali veb-saytlarning "orqa tomoni"ni yaratishda ishlatiladi.
  • ✅ Ko'p qirrali: Sun'iy intellekt, ma'lumotlar tahlili, avtomatlashtirish, o'yinlar — hammasini Python bilan qilish mumkin.
  • ✅ Katta hamjamiyat: Dunyoda eng ko'p ishlatiladigan tillardan biri. Muammo bo'lsa, yechim topish oson.

🎯 Backend nima? Veb-sayt ikki qismdan iborat: Frontend (foydalanuvchi ko'radigan qism) va Backend (serverda ishlaydigan, ma'lumotlarni boshqaradigan qism). Python asosan Backendda ishlatiladi.

1.4 Birinchi "Hello World" dasturi va print() funksiyasi

Har bir dasturchi o'z yo'lini "Hello, World!" dasturidan boshlaydi. Bu an'ana 1970-yillardan beri davom etib kelmoqda. Keling, biz ham shundan boshlaylik! 

# Bu sizning birinchi dasturingiz!

print("Salom, Dunyo!")
print("Men Python dasturchisiman! 🐍")

💡 print() nima qiladi? print() — bu Pythonning o'rnatilgan funksiyasi bo'lib, u qavs ichidagi ma'lumotni ekranga chiqaradi. "Print" inglizchadan "chop etish" degani.

print() funksiyasi sirlari

# Bir nechta qiymatni vergul bilan chiqarish
print("Ism:", "Ali", "Yosh:", 25)
# Natija: Ism: Ali Yosh: 25

# sep parametri — oraliq belgini belgilaydi
print("Python", "Django", "FastAPI", sep=" | ")
# Natija: Python | Django | FastAPI

# end parametri — oxirida nima qo'yilishini belgilaydi
print("Salom", end=" ")
print("Dunyo!")
# Natija: Salom Dunyo! (bir qatorda)

# \n — yangi qatorga o'tish
print("Birinchi qator\nIkkinchi qator")
# Natija:
# Birinchi qator
# Ikkinchi qator

# \t — tab (bo'sh joy) qo'yish
print("Ism:\tAli")
print("Yosh:\t25")

⚠️ Eslatma: print() funksiyasi har safar oxirida avtomatik yangi qatorga o'tadi. end="" parametri bilan buni o'zgartirish mumkin.

1.6 O'zgaruvchilar (variables) va nomlash qoidalari

O'zgaruvchi — bu ma'lumotni saqlash uchun "quti". Tasavvur qiling, sizda turli xil qutilar bor: birida ismingiz, birida yoshingiz, birida manzilingiz saqlanadi. Har bir qutining nomi bor — bu o'zgaruvchi nomi. 

# O'zgaruvchi yaratish — juda oson!

yosh = 25             # Butun son (integer)
boy = 1.75            # Kasr son (float)
talaba = True         # Mantiqiy qiymat (boolean)

print(ism)            # Ali
print(yosh)           # 25
print(boy)            # 1.75
print(talaba)         # True

PEP-8 nomlash qoidalari

PEP-8 — bu Python kodini yozish bo'yicha rasmiy uslub qo'llanmasi. O'zgaruvchilarni nomlashda quyidagi qoidalarga rioya qiling: 

# ✅ TO'G'RI nomlash:
ism = "Ali"                    # kichik harf, tushunarli
talaba_yoshi = 20              # snake_case (pastki chiziq bilan)
universitet_nomi = "TATU"      # tushunarli va aniq

# ❌ NOTO'G'RI nomlash:
Ism = "Ali"                    # Katta harf bilan boshlanmasligi kerak
talabaYoshi = 20               # camelCase — Python'da tavsiya etilmaydi
x = "Ali"                      # tushunarsiz, qisqa nom
2yosh = 20                     # raqam bilan boshlanmasligi kerak!
ism-familiya = "Ali Vali"      # defis (-) ishlatilmaydi!
class = "5-sinf"               # Python kalit so'zlarini ishlatmang!

# Python kalit so'zlari (bularni o'zgaruvchi nomi qilib bo'lmaydi):
# False, None, True, and, as, assert, break, class, continue,
# def, del, elif, else, except, finally, for, from, global,
# if, import, in, is, lambda, nonlocal, not, or, pass, raise,
# return, try, while, with, yield

💡 Qoida: O'zgaruvchi nomi harf yoki pastki chiziq (_) bilan boshlanishi kerak. Raqam bilan boshlash mumkin emas. Faqat harflar, raqamlar va pastki chiziq ishlatiladi.

1.7 Asosiy ma'lumot turlari (int, float, str, bool)

Pythonda har bir qiymatning turi (type) bor. Bu turlar kompyuterga ma'lumot bilan qanday ishlash kerakligini aytadi. Keling, 4 ta asosiy turni ko'rib chiqaylik: 

# 1️⃣ int (Integer) — Butun sonlar
yosh = 25
yil = 2025
harorat = -10
print(type(yosh))    # <class 'int'>

# 2️⃣ float — Kasr (o'nli) sonlar
boy = 1.75
narx = 99.99
pi = 3.14159
print(type(boy))     # <class 'float'>

# 3️⃣ str (String) — Matnlar
ism = "Ali"
shahar = 'Toshkent'
xabar = """Bu ko'p qatorli
matn bo'lishi mumkin"""
print(type(ism))     # <class 'str'>

# 4️⃣ bool (Boolean) — Mantiqiy qiymatlar (True yoki False)
talaba = True
admin = False
print(type(talaba))  # <class 'bool'>

# type() funksiyasi — ma'lumot turini aniqlaydi
a = 42
b = 3.14
c = "Python"
d = True

print(type(a))  # <class 'int'>
print(type(b))  # <class 'float'>
print(type(c))  # <class 'str'>
print(type(d))  # <class 'bool'>

💡 Esda tuting: Matn doimo qo'shtirnoq " " yoki ' ' ichida yoziladi. Raqamlar qo'shtirnoqsiz yoziladi. True va False katta harf bilan yoziladi!

# Ma'lumot turlarini solishtirish
print("100")    # Bu — matn (str)
print(100)      # Bu — son (int)
print(100.0)    # Bu — kasr son (float)

# Ularning turi har xil!
print(type("100"))   # <class 'str'>
print(type(100))     # <class 'int'>
print(type(100.0))   # <class 'float'>

# True va False — bu aslida 1 va 0
print(int(True))   # 1
print(int(False))  # 0

# 1) Yagona qo'shtirnoq bilan

2.3 Indekslash va qirqib olish (Slicing)

Har bir matndagi belgining o'z indeksi (raqami) bor. Indekslash 0 dan boshlanadi! 

#        0123456789...
matn = "Python Dasturlash"

# 📌 Indekslash — bitta belgini olish
print(matn[0])    # P (birinchi belgi)
print(matn[1])    # y
print(matn[5])    # n
print(matn[-1])   # h (oxirgi belgi — manfiy indeks!)
print(matn[-2])   # a (oxirdan ikkinchi)

# 📌 Slicing — matnni qirqib olish
# matn[boshlanish:tugash] — tugash indeksi kirmaydi!
print(matn[0:6])   # Python (0,1,2,3,4,5 — 6 kirmaydi)
print(matn[7:17])  # Dasturlash
print(matn[:6])    # Python (boshidan 6 gacha)
print(matn[7:])    # Dasturlash (7 dan oxirigacha)
print(matn[:])     # Python Dasturlash (hammasi)

# 📌 Qadam bilan slicing
# matn[boshlanish:tugash:qadam]
print(matn[::2])   # Pto atsulh (har 2-belgini oladi)
print(matn[::-1])  # halsuratsaD nohtyP (TESKARI!)

# 📌 Amaliy misol
email = "foydalanuvchi@gmail.com"
print(email[:13])    # foydalanuvchi
print(email[14:])    # gmail.com
print(email[-9:])    # gmail.com

💡 Muhim: Indekslash 0 dan boshlanadi. Slicing'da tugash indeksi kirmaydi. Manfiy indeks — oxirdan boshlab sanaydi. [::-1] — matnni teskari qiladi!

2.4 Eng ko'p ishlatiladigan String metodlari

String metodlari — bu matn ustida turli amallar bajarish uchun tayyor funksiyalar. Ularni matn.metod() ko'rinishida ishlatamiz. 

matn = "Salom Dunyo"

# 📌 upper() — BARCHA HARFLARNI KATTA qiladi
print(matn.upper())        # SALOM DUNYO

# 📌 lower() — barcha harflarni kichik qiladi
print(matn.lower())        # salom dunyo

# 📌 title() — Har Bir So'zning Bosh Harfini Katta Qiladi
print("python dasturlash".title())  # Python Dasturlash

# 📌 capitalize() — Faqat birinchi harfni katta qiladi
print("python dasturlash".capitalize())  # Python dasturlash

# 📌 replace() — matnning bir qismini almashtiradi
print(matn.replace("Dunyo", "Python"))  # Salom Python
print(matn.replace("o", "0"))           # Sal0m Duny0

# 📌 split() — matnni bo'laklarga ajratadi (ro'yxat qaytaradi)
gap = "Python juda zo'r til"
sozlar = gap.split()       # Bo'sh joy bo'yicha ajratadi
print(sozlar)              # ['Python', 'juda', "zo'r", 'til']

sanalar = "2025-01-15"
qismlar = sanalar.split("-")
print(qismlar)             # ['2025', '01', '15']

# 📌 strip(), lstrip(), rstrip() — bo'sh joylarni olib tashlaydi
matn2 = "   Salom   "
print(matn2.strip())       # "Salom" (ikki tomondan)
print(matn2.lstrip())      # "Salom   " (chapdan)
print(matn2.rstrip())      # "   Salom" (o'ngdan)

# 📌 count() — belgi/so'z sonini sanaydi
print("banana".count("a"))  # 3

# 📌 find() — belgi indeksini topadi (topilmasa -1 qaytaradi)
print("Python".find("t"))   # 2
print("Python".find("z"))   # -1

# 📌 startswith() va endswith() — boshlanishi/tugashini tekshiradi
print("hello.py".endswith(".py"))    # True
print("hello.py".startswith("hel"))  # True

# 📌 len() — matn uzunligini (belgi sonini) qaytaradi
print(len("Python"))  # 6

2.5 Matn formatlash: f-string va .format()

O'zgaruvchilarni matn ichiga joylashtirishning bir necha usuli bor. Eng zamonaviy va qulay usul — f-string. 

ism = "Ali"
yosh = 25
shahar = "Toshkent"

# ✅ 1-usul: f-string (Eng tavsiya etilgan — Python 3.6+)
xabar = f"Mening ismim {ism}, yoshim {yosh}, {shahar}da yashayman"
print(xabar)
# Natija: Mening ismim Ali, yoshim 25, Toshkentda yashayman

# f-string ichida hisoblash ham mumkin!
a = 10
b = 20
print(f"{a} + {b} = {a + b}")         # 10 + 20 = 30
print(f"Keyingi yil men {yosh + 1} yoshda bo'laman")

# ✅ 2-usul: .format() (Eski usul, lekin hali ishlatiladi)
xabar2 = "Mening ismim {}, yoshim {}".format(ism, yosh)
print(xabar2)

# Nomli .format()
xabar3 = "Ism: {name}, Yosh: {age}".format(name=ism, age=yosh)
print(xabar3)

# ❌ 3-usul: + bilan birlashtirish (noqulay)
xabar4 = "Mening ismim " + ism + ", yoshim " + str(yosh)
print(xabar4)

# 📌 f-string formatlash imkoniyatlari
pi = 3.14159265
print(f"Pi: {pi:.2f}")      # Pi: 3.14 (2 xona kasr)
print(f"Pi: {pi:.4f}")      # Pi: 3.1416 (4 xona kasr)

son = 42
print(f"Son: {son:0>5}")    # Son: 00042 (chapdan 0 bilan to'ldirish)
print(f"Son: {son:*>10}")   # Son: ********42

🎯 Tavsiya: Har doim f-string ishlating! Bu eng o'qilishi oson, eng tez va eng zamonaviy usul.

2.6 Foydalanuvchidan ma'lumot qabul qilish: input()

input() funksiyasi foydalanuvchidan klaviatura orqali ma'lumot qabul qiladi. Dastur foydalanuvchi yozib bo'lguncha to'xtab turadi (Enter bosilguncha). 

# input() — har doim STRING (matn) qaytaradi!
ism = input("Ismingiz nima? ")
print(f"Salom, {ism}!")

yosh = input("Yoshingiz nechchi? ")
print(f"Siz {yosh} yoshdasiz")

# input() har doim str qaytaradi, shuning uchun:
yosh = int(input("Yoshingiz nechchi? "))  # int ga o'tkazdik
print(f"Keyingi yil siz {yosh + 1} yoshda bo'lasiz")

# Amaliy misol: Kalkulyator
son1 = float(input("Birinchi son: "))

print(f"{son1} + {son2} = {son1 + son2}")
print(f"{son1} - {son2} = {son1 - son2}")

print(f"{son1} / {son2} = {son1 / son2}")

⚠️ MUHIM: input() HAR DOIM str (matn) qaytaradi! Agar son kerak bo'lsa, int() yoki float() bilan o'rab oling.

        return natija
    return wrapper

# 📌 raise — sun'iy xatolik chaqirish
def yosh_tekshirish(yosh):
    if yosh < 0:
        raise ValueError("Yosh manfiy bo'lishi mumkin emas!")
    if yosh > 150:
        raise ValueError("Yosh juda katta!")
    return f"Yoshingiz: {yosh}"

# 📌 Custom Exception — o'z xatolik sinfingiz
class ParolXatosi(Exception):
    """Parol bilan bog'liq xatoliklar"""
    pass

class ParolQisqa(ParolXatosi):
    """Parol juda qisqa"""
    pass

def parol_tekshirish(parol):
    if len(parol) < 8:
        raise ParolQisqa("Parol kamida 8 ta belgi bo'lishi kerak!")
    if not any(c.isdigit() for c in parol):
        raise ParolXatosi("Parolda raqam bo'lishi kerak!")
    return "✅ Parol qabul qilindi"

try:
    parol_tekshirish("abc")
except ParolQisqa as e:
    print(f"Parol xatosi: {e}")

Modul — bu Python fayli (.py) bo'lib, unda funksiyalar, sinflar va o'zgaruvchilar bo'ladi. Modullar kodni qayta ishlatish imkonini beradi.

# "hisoblash.py" fayli yarating:

# === hisoblash.py ===
"""Hisoblash moduli — arifmetik amallar"""

PI = 3.14159

def qosh(a, b):
    return a + b

def ayir(a, b):
    return a - b

def kopaytir(a, b):
    return a * b

def bol(a, b):
    if b == 0:
        raise ValueError("Nolga bo'lish mumkin emas!")

from hisoblash import qosh, kvadrat

from hisoblash import qosh as q
